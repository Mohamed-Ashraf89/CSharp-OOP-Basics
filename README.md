# C# Object-Oriented Programming (OOP) Basics 🚀

This repository is a practical collection of C# projects and exercises designed to master the core principles of OOP and .NET fundamentals.

---

## 📂 Projects & Features

### 1. Banking System (Bank Account Management) 🏦
A robust implementation of a bank account system that demonstrates high-level class architecture and data security.
* **Key OOP Concepts Applied:**
    * **Static Members:** Used `static` properties to track `TotalAccounts` and a global `BankName`.
    * **Constructor Overloading:** Features Default, Parameterized, and Copy constructors for flexible object initialization.
    * **Static Constructor:** Used to initialize shared class data like `BankName` once per application lifecycle.
    * **Encapsulation:** Protected sensitive data (Balance, Account Numbers) using private setters and logic-based methods (`Deposit`, `Withdraw`).
    * **Auto-ID Generation:** Implemented a private static field to safely generate unique account numbers.

### 2. Extension Methods Utility 🛠️
The initial project demonstrating how to extend built-in .NET types with custom functionality.
* **Features:**
    * `IsGreaterThan`: Enhanced integer comparison.
    * `IsNumber`: Safe string-to-integer validation.
    * `CountWords`: Practical sentence analysis logic.
