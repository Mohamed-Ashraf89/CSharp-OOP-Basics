# CSharp-OOP-Basics
Learning OOP With C#

A simple C# Console Application that demonstrates the power of Extension Methods. This project allows users to compare numbers, validate strings, and count words in a sentence.

# Features

IsGreaterThan: Compares two integers in a more readable way.

IsNumber: Checks if a string can be safely converted to an integer.

CountWords: Calculates the number of words in a given sentence.

# How to use
```
int myNumber = 10;
if (myNumber.IsGreaterThan(5)) {
    // Your logic here
}
```

I built this project while studying OOP and C# Fundamentals to practice creating reusable code.
